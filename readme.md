# fast-integral
One line JS library to calculate integral of axⁿ type equations. 

## Usage
```diff
import integral from 'fast-integral';

// Calculate integral of 2x² between 0 and 4
console.log(integral(2, 2, 0, 4));
// > 42.6666666667
```
